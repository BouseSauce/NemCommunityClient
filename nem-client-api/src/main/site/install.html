Introduction

It is important to note, that Oracle not yet fully supports the automatic installation of Java 8.
In case that you are directed to a Java download for Java 7, please ignore that version and download the Java 8 from the following Java page.

Official Oracle Java8 Download page

Please pay also attention that you select the correct version (64Bit or 32Bit). The bit version is not dependent from the Operating Version you are using but from the browser. If the browser you're using is 32-bit, then please install a 32-bit version of Java8.

Some browsers will show a dialog whether to download or to start a program. Chrome is one of those browsers, please follow the solution here.

If you are on Mac, even if the application is signed by an official certificate, security settings are blocking JavaWebStart.
Unfortunately, you have to adjust the security settings. Please read this article.
